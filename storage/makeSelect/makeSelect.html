• 최초생성

    $('select').jqListBox();

  • ajax로 option리스트 변경 후 사용해야함

    ajax로 option리스트를 변경했을 경우 해당 기본셀렉트에 .refresh() 로 갱신

    $('#sel1').empty().html(dynamicOption).refresh();

  • title 사용가능 (미지정시 option의 텍스트를 title로 추가)

    <option />에 title입력 시 jQList에 title추가 됨

    <option value="http://www.nate.com" title="새창">nate</option>

  • select의 id값이 data-origin 값으로 들어감

    시스템셀렉트의 ID값이 jqList의 data-origin으로 들어가서, 현재 jqList의 기본셀렉트를 탐색하는 등의 활용

    var selectID = $('.jqListBox').data('origin')

  • data-class 속성으로 개별 클래스 지정가능

    스타일이 다른 셀렉트를 만들 경우 해당 그룹끼리 공통으로 data-class에 값을 넣어 스타일 지정 등 의 활용

    <select data-class="select_termList"> -> <span class="jqListBox select_termList">

  • data-maxheight 속성 : 리스트 스크롤 처리

    data-maxheight에 설정한 값이 List의 max-height의 px값으로 지정되며 스크롤처리 가능

    <select data-maxheight="150">

  • data-width 속성 : 생성된 리스트 넓이 지정 (미지정시 시스템셀렉트의 넓이로 자동세팅)

    data-width에 설정한 값이 jqListBox의 width px값으로 고정

    <select data-width="2000">
