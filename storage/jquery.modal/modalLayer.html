jqModal - layerPopup

	jqModal_min.js 임포트

	소스보기  내용의 마크업 구조 기준으로 레이어HTML작성

	$('#list').find('a').click(function(e){
		var target = $(this).attr('href');
		$(this).jqModal(target);
		e.preventDefault();
	});

	$(호출객체 {a tag} ).jqModal('오픈 할 레이어팝업 아이디 ex) #ly-login  {string} ');

	버튼에 data-dismiss="modal" 을 셋팅하여 닫기기능을 추가하는 것 외에,
	강제로 이벤트를 발생하고자 할 경우, $.jqModal_close(); 를 실행
	ex )데이터를 전송할 경우 성공했을 경우 콜백으로 $.jqModal_close(); 을 실행
